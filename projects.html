CS61A Projects

hog: a dice game

in hog, two players alternate turns trying to be the first to end a turn with at least 100 total points. on each turn, the current player chooses some number of dice to roll, up to 10. that player's score for the turn is the sum of the dice outcomes. read the rules and the spec: project page (Description copied from that page)

cats: a typing test

cats, or the cs61a autocorrected typing software, is an autocorrected typing speed test! read the rules and the spec: project page

ant: a tower defense game

ants is a game that is based off the famous popcap game, plants vs zombies. it is a tower defense game that pits ants which throw projectiles against bees which try to invade. read the rules and the spec: project page

scheme: a fully functinonal interpreter

scheme is a fully functional and tail recursive scheme interpreter. scheme is a functional dialect of lisp, or list processing languages. read the rules and the spec: project page
